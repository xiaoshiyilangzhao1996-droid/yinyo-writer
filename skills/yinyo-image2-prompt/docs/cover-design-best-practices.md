## 0.5 Image2 文字海报 / 公众号封面实战原则（2026-05-01）

适用于 `image2-typography-poster` 与公众号文章主封面。

### 核心判断

封面首先不是“好看”，而是让读者在缩略图里记住文章的核心词。

优先级：

1. **目标字词准确可读**：无错字、漏字、伪字、乱码。
2. **目标字词最大最醒目**：人物、机器人、手指、背景都不能抢主体。
3. **主题贴合度高于品牌色相似度**：像 DeepSeek 不如让人记住“视觉原语”。
4. **一个精准隐喻胜过十个科技装饰**：坐标、point、bounding box、路径线比普通蓝色电路更贴题。
5. **先看手机缩略图**：大图细节好看不等于公众号封面有效。

### DeepSeek视觉原语案例

目标词：`DeepSeek视觉原语`。

- v1：红黑 / 灰白红点缀，建立“空间指认系统”。核心元素为视觉坐标、point、bounding box、路径线、指向、Agent 定位世界。
- v2：DeepSeek 蓝色科技风，深蓝、冷蓝、青蓝、透明 UI、网格、扫描线。
- 最终选择 v1：因为 v1 更贴合文章主题，v2 更像普通 AI 科技封面。

### 生成后验收

- [ ] 缩略图里能一眼读出目标词。
- [ ] 目标词没有错字、漏字、多字、伪字。
- [ ] 画面没有水印、Logo、二维码、多余说明文字。
- [ ] 没有人物/机器人/手指抢过文字。
- [ ] 风格服务文章主题，而不是只服务品牌色。
- [ ] 如生成备选风格，最终选择按“主题记忆点”而不是“像不像某品牌”。

## 核心内容

### 1. 爆款封面的核心特质

#### 1.1 视觉冲击力
- **第一印象**：封面图在0.5秒内决定用户是否点击
- **高对比度**：颜色对比鲜明，快速吸引眼球
- **清晰焦点**：有明确的视觉重心，不要让用户思考"我看什么"
- **简洁有力**：信息不过载，突出核心内容

#### 1.2 信息传达效率
- **一目了然**：3秒内明白文章主题
- **文字可读**：字号够大、对比度高、字体清晰
- **关键信息**：标题、数字、核心观点一眼可见
- **情感共鸣**：引发用户的好奇、焦虑、兴趣等情绪

#### 1.3 品牌辨识度
- **风格统一**：同一公众号的封面风格保持一致
- **色彩体系**：使用固定的主色调（2-3种）
- **字体选择**：标题字体统一
- **元素重复**：使用固定的视觉元素（如边框、图标、logo）

#### 1.4 移动端优化
- **尺寸适配**：900×383像素（2.35:1比例）
- **清晰度**：文字和图片在手机上清晰可辨
- **信息层级**：主标题 > 副标题 > 装饰元素
- **留白充足**：避免元素过于拥挤

### 2. 封面设计的关键要素

#### 2.1 构图原则

**三分法构图**：
- 将画面分为3×3的九宫格
- 重要元素放在交叉点或分割线上
- 主标题通常放在左上或中上位置

**中心构图**：
- 核心元素居中
- 对称或平衡的布局
- 适合强调单一主题

**对角线构图**：
- 元素沿对角线分布
- 创造动感和张力
- 适合展示对比或冲突

**留白原则**：
- 不要填满整个画面
- 重要元素周围留出足够空间
- 提升高级感和呼吸感

#### 2.2 色彩策略

**主色调选择**：
- 科技类：蓝色、紫色、黑色（科技感）
- 情感类：粉色、浅紫、米色（温暖）
- 职场类：深蓝、黑色、灰色（专业）
- 教育类：绿色、蓝色、橙色（成长）

**对比度要求**：
- 文字与背景对比度 > 4.5:1
- 避免使用浅色文字配浅色背景
- 深色文字配浅色背景，或浅色文字配深色背景

**配色方案**：
- 单色搭配：同一色相的不同明度
- 近似色：色相环上相邻的颜色
- 互补色：色相环上相对的颜色（强烈对比）
- 三色搭配：一个主色+两个辅助色

**配色示例**：
```
科技风：
- 主色：#2B5B84（深蓝）
- 辅色：#F5F5F5（浅灰）
- 强调色：#FF6B6B（橙红）

情感风：
- 主色：#FFB6C1（浅粉）
- 辅色：#FFF5EE（米白）
- 强调色：#8B4513（深棕）

职场风：
- 主色：#2C3E50（深蓝灰）
- 辅色：#ECF0F1（浅灰）
- 强调色：#E74C3C（红）

教育风：
- 主色：#27AE60（绿）
- 辅色：#FEF9E7（浅黄）
- 强调色：#E67E22（橙）
```

#### 2.3 文字排版

**字号选择**：
- 主标题：48-64px（占据画面1/4-1/3高度）
- 副标题：28-36px（主标题的1/2大小）
- 装饰文字：16-20px（如"必读"、"推荐"等标签）

**字体选择**：
- 粗体字：用于主标题，突出重点
- 常规字体：用于副标题，保持可读性
- 衬线字体：适合正式、专业风格
- 无衬线字体：适合现代、简洁风格

**文字位置**：
- 主标题：左上、中上、居中
- 副标题：主标题下方或右侧
- 装饰标签：右上或左上角

**文字间距**：
- 字间距：字号的5-10%
- 行间距：字号的120-150%
- 与其他元素间距：20-40px

**文字对比**：
- 纯色背景+纯色文字
- 渐变背景+纯色文字
- 图片背景+文字阴影/描边

#### 2.4 视觉元素

**图标使用**：
- 简洁的图标：放大、星标、对勾等
- 风格统一：所有图标使用相同风格
- 大小适中：不要喧宾夺主

**边框和分隔**：
- 使用边框突出重要区域
- 分隔线区分不同信息
- 保持简约，不要过度装饰

**Logo和水印**：
- 位置：右下角或右上角
- 大小：不超过画面的1/10
- 透明度：70-80%

### 3. 不同赛道的封面风格

#### 科技赛道

**头部博主**：量子位、新智元、机器之心

**封面特质**：
- **色彩**：深蓝、黑色、科技蓝、霓虹色
- **构图**：科技图表、产品截图、抽象图形
- **文字**：粗体大标题，关键词突出
- **元素**：电路板、芯片、数据流、AI图标

**典型封面样式**：
```
背景：深蓝色渐变
主标题：白色粗体 "GPT-4重大升级"
副标题：浅灰色 "性能提升30%"
元素：电路板纹理背景
装饰：右上角"重磅"标签
```

**生成指导**：
- 使用"科技感"、"未来感"、"AI技术"等关键词
- 背景使用深色渐变或科技纹理
- 主标题使用白色或亮色粗体字
- 添加科技元素：芯片、电路板、数据图表

**关键词示例**：
- 主题：AI大模型
- 风格：科技感、未来感、专业
- 构图：科技图表背景、标题居中偏左
- 色彩：深蓝+霓虹蓝+白色
- 文字：粗体大标题"大模型战争"，副标题"谁能笑到最后？"

#### 情感赛道

**头部博主**：十点读书、夜听、有书、洞见

**封面特质**：
- **色彩**：粉色、浅紫、米色、温暖色调
- **构图**：插画、风景、人物特写、抽象图案

- 背景使用柔和渐变或花卉、风景
- 主标题使用深色粗体字，营造稳重感
- 添加温暖元素：花朵、星光、爱心

**关键词示例**：
- 主题：情感治愈
- 风格：温暖、治愈、柔和
- 构图：花卉背景、标题居中

### 4. 封面生成的具体要求

#### 4.1 尺寸规格

**标准尺寸**：
- 宽度：900px
- 高度：383px
- 比例：2.35:1

**导出格式**：
- 格式：PNG（保持透明度）
- 质量：最高质量（100%）
- 分辨率：72-150dpi（屏幕显示）

#### 4.2 生成步骤

**步骤1：确定封面主题**
- 提炼文章核心主题
- 确定目标读者
- 选择合适的赛道风格

**步骤2：选择配色方案**
- 根据赛道选择主色调
- 确定背景色和文字色
- 检查对比度是否达标

**步骤3：设计构图**
- 确定主标题位置
- 安排副标题和装饰元素
- 留出足够留白

**步骤4：生成封面图**
- 使用详细的描述词
- 明确构图、色彩、文字
- 生成后检查可读性和美观度

**步骤5：优化调整**
- 检查文字是否清晰可读
- 调整色彩和对比度
- 确保移动端显示效果

### 5. 封面优化技巧

#### 5.1 提升视觉冲击力

**使用高饱和度色彩**：
- 主色饱和度提高10-20%
- 但不要过度，保持专业感

**增加对比度**：
- 文字与背景对比度 > 4.5:1
- 使用深色文字配浅色背景，反之亦然

**简化构图**：
- 突出核心元素
- 去除不必要的装饰
- 保持画面简洁

#### 5.2 提升信息传达

**优化标题**：
- 字号足够大（48-64px）
- 使用粗体
- 确保在手机上清晰可读

**突出关键词**：
- 数字、重要概念使用不同颜色
- 副标题补充核心信息

**使用标签**：
- "重磅"、"必读"、"推荐"等标签
- 放在右上角显眼位置
- 使用对比色

#### 5.3 移动端优化

**文字大小**：
- 主标题最小48px
- 副标题最小28px

**元素间距**：
- 文字与其他元素间距 ≥ 20px
- 留出足够呼吸空间

**清晰度**：
- 导出为PNG格式
- 使用最高质量

### 6. 常见问题与解决方案

#### 问题1：文字看不清楚

**原因**：
- 文字颜色与背景对比度低
- 字号太小
- 背景过于复杂

**解决方案**：
- 增加文字与背景的对比度
- 增大字号（主标题≥48px）
- 简化背景，使用纯色或简单渐变

#### 问题2：封面过于复杂

**原因**：
- 元素过多
- 色彩过多
- 信息过载

**解决方案**：
- 突出1-2个核心元素
- 限制颜色数量（最多3种）
- 删除不必要的装饰

#### 问题3：风格不统一

**原因**：
- 每次生成使用不同的风格
- 色彩体系混乱
- 字体不一致

**解决方案**：
- 为公众号制定统一的封面风格指南
- 固定使用2-3种主色调
- 使用相同的字体和字号规范

#### 问题4：移动端显示不佳

**原因**：
- 尺寸不符合900×383
- 文字过小
- 元素过于拥挤

**解决方案**：
- 严格使用900×383像素
- 主标题≥48px，副标题≥28px
- 增加元素间距，留出留白

### 7. 更多封面设计风格

#### 7.1 插画风格

**特点**：
- 手绘感、艺术性强
- 适合情感、教育、美食类
- 温暖、亲切

**设计要点**：
- 使用插画背景
- 文字叠加在插画上，使用阴影或描边
- 保持插画简洁，不喧宾夺主

**示例**：
```
背景：手绘城市插画
主标题：白色粗体，带黑色描边
副标题：浅色，叠加在插画空白处
风格：文艺、温暖
```

#### 7.2 照片风格

**特点**：
- 真实感强
- 适合旅游、美食、时尚类
- 现代感、品质感

**设计要点**：
- 使用高质量照片背景
- 文字使用半透明背景框
- 或在照片空白处添加文字

**示例**：
```
背景：风景照片
文字区域：半透明白色背景框（80%透明度）
主标题：黑色粗体
副标题：深灰色
风格：现代、真实
```

#### 7.3 极简风格

**特点**：
- 纯色背景，无装饰
- 适合科技、职场、教育类
- 专业、简洁

**设计要点**：
- 使用纯色或简单渐变背景
- 文字居中或左对齐
- 使用大字号，突出标题

**示例**：
```
背景：纯深蓝色
主标题：白色粗体，居中
副标题：浅灰色，主标题下方
风格：极简、专业
```

#### 7.4 渐变风格

**特点**：
- 色彩过渡柔和
- 适合情感、教育、科技类
- 现代、时尚

**设计要点**：
- 使用2-3色渐变背景
- 选择对比度高的颜色组合
- 文字使用白色或深色

**示例**：
```
背景：紫色到蓝色渐变
主标题：白色粗体
副标题：浅色半透明
风格：现代、时尚
```

#### 7.5 扁平化风格

**特点**：
- 简洁的几何图形
- 适合科技、教育类
- 现代、清晰

**设计要点**：
- 使用简单的几何图形作为装饰
- 配色明亮，对比度高
- 文字清晰简洁

**示例**：
```
背景：浅灰色
装饰：扁平化几何图形（三角形、圆形）
主标题：深色粗体
副标题：橙色
风格：扁平化、现代
```

### 8. 字体选择详细指导

#### 8.1 字体分类

**衬线字体**（Serif）：
- 特点：笔画末端有装饰线
- 风格：正式、专业、传统
- 适用：职场、教育、金融类
- 示例：宋体、楷体、Georgia

**无衬线字体**（Sans-serif）：
- 特点：笔画简洁，无装饰
- 风格：现代、简洁、易读
- 适用：科技、情感、美食类
- 示例：黑体、微软雅黑、Arial

**手写体**（Script）：
- 特点：模拟手写风格
- 风格：温暖、亲切、个性
- 适用：情感、美食、教育类
- 示例：行楷、手写体

#### 8.2 不同赛道的字体选择

**科技赛道**：
```
主标题：无衬线粗体（黑体、微软雅黑）
副标题：无衬线常规（微软雅黑、思源黑体）
风格：现代、科技、简洁
```

**情感赛道**：
```
主标题：衬线粗体（宋体）或优雅无衬线（方正兰亭）
副标题：衬线常规（宋体）或手写体（行楷）
风格：优雅、温暖、细腻
```

**职场赛道**：
```
主标题：无衬线粗体（微软雅黑 Bold、思源黑体 Bold）
副标题：无衬线常规（微软雅黑、思源黑体）
风格：专业、现代、简洁
```

**教育赛道**：
```
主标题：无衬线粗体（微软雅黑 Bold）或友好字体（方正卡通）
副标题：无衬线常规（微软雅黑）或手写体（方正楷体）
风格：友好、亲切、活泼
```

#### 8.3 字号规范

**主标题**：
- 科技类：56-64px
- 情感类：52-60px
- 职场类：54-62px
- 教育类：50-58px

**副标题**：
- 科技类：28-34px
- 情感类：30-36px
- 职场类：28-34px
- 教育类：32-38px

**标签文字**：
- 字号：16-20px
- 位置：右上角或左上角
- 背景：对比色块

## 约束与注意事项
- 封面尺寸必须严格为900×383像素（2.35:1比例）
- 文字与背景对比度必须 > 4.5:1
- 导出格式必须使用PNG
- 同一公众号的封面风格要保持统一
- 避免使用过度花哨的效果，保持专业简洁
- 定期分析爆款封面，不断优化设计
- 注意版权问题，使用正版图片和字体
- 生成前检查移动端显示效果
