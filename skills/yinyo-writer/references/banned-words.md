# 禁用词表与推荐词表

> 从 SKILL.md 外移，供 L1 质检和写作时查阅。
> Aligned with yinyo-writer v4.0.3+

## 高频踩雷词（绝对禁用）

| 禁用词 | 替代表达 |
|---|---|
| 说白了 | 坦率的讲、其实就是 |
| 意味着什么/这意味着 | 所以呢、那结果会怎样呢 |
| 本质上 | 说到底、其实 |
| 换句话说 | 也就是说、你想想看 |
| 综上所述 | （删掉，用具体的回扣句） |
| 首先...其次...最后 | 用自然的转场词 |
| XXX不是XXX，而是XXXX | 拆成两句具体判断，或直接说正向结论 |
| 不难发现/值得注意的是 | （直接说内容） |
| 题外话 | （直接删掉，该聊就聊，不聊就不写） |
| 今天先到这里 | （删掉，结尾就是结论，没有缓冲地带） |
| 这个话题改天再聊 | （删掉，要聊就现在聊完） |
| 让我们来看看 | （删掉） |
| 在当今...的时代 | （从具体事件切入，不用这种） |
| 本文将从以下几个方面展开 | 从具体冲突或判断切入 |
| 具有重要意义 | 说清楚具体改变了谁的什么处境 |
| 赋能/闭环/抓手/底层逻辑 | 换成具体动作、路径或后果 |
| 革命性/颠覆性/划时代 | 用证据、案例或边界说明强度 |
| 未来已来/拥抱变化/成为更好的自己 | 回到具体的人类处境或行动 |
| 时代洪流/历史车轮/人类命运 | 没有具体场景承接时删掉 |
| 欲知后事如何 | （删掉，避免章回体过度） |
| 话说/且说/列位看官 | （删掉，避免仿古说书腔） |
| 花叔（作为风格标签） | 统一称作「yinyo红色组件」或「yinyo」；任何排版样式的参考命名都不允许出现「花叔」字样 |

## 推荐口语词组

这些词组自带"真人感"，自然使用，不要硬塞：

**转场过渡**：怎么说呢、其实吧、你想想看、回到这块、顺着上面的再聊聊、这块需要注意一下

**表达判断**：我有时候觉得、我始终坚信、这话听着有点刺耳但、真正的问题在于

**历史叙事转场**：事情到这里开始变味了、真正的分水岭来了、问题出在这里、这时候新角色登场、功劳要认，代价也要看

**承认和自嘲**：说实话我也不确定、可能有些想法还不成熟、我自己也在摸索、愚钝如我

**拉近距离**：很多朋友可能不知道、可能有小伙伴纳闷、大家也都知道

**情绪表达**：给我整懵了、太离谱了、一时间无语凝噎、你敢信？？？
